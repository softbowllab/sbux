SBUx v2.6 Components

SBUx v2.6 는 SI 및 Application 개발자들이 퀄리티 있는 화면을 만들기 위해 필요한 다양한 기능등을 쉽게 사용할 수 있도록 지원하는 UI/UX 컴포넌트입니다.

IE, 크롬, 파이어폭스, 사파리등의 PC 브라우저와 HTML5가 지원되는 최신의 모바일 환경(스마트폰, 태블릿PC)에서 사용이 가능합니다.

Browser Support

크롬, 파이어폭스, 사파리, IE 9 이상 (SBChart IE 10+)을 권장합니다.

Features

  • HTML5 웹 표준 (W3C, RFC, ECMA 등)을 준수합니다.

    - W3C Markup Validation Service

    - W3C CSS Validation Service

    - RFC 8259/7159, ECMA-404 JSON

  • 개발자가 웹 개발에 부담을 느끼지 않게 쉽게 만들었습니다.

    - HTML Markup 또는 JavaScript 의 익숙하신 형태로 개발이 가능합니다.

    - 컴포넌트 내 Custom 속성을 제한없이, 필요하신 정보를 마음대로 입력 가능합니다.

    - 컴포넌트 그룹화로 일괄적으로 기능을 처리할 수 있습니다.

  • 25개종 41개 유형의 다양한 컴포넌트가 준비되어 있습니다.

    - Control Components

    - Input, Textarea, Label, Button, Datepicker,Select, Checkbox Select, Dropdown List,Listbox, Checkbox, Radio

    - Navigation Components

    - Contextmenu, Menu, Side Menu

    - Layout Components

    - TabStrip, Carousel, Accordion

    - Popups Components

    - Window, Modal

    - Notification Components

    - Tooltip, Popover, Alert, ProgressBar

    - Data Components

    - Tree, Datastore, SBGrid

    - Visualization Components

    - SBChrat

  • 서버 데이터 활용이 간편합니다.

    - 서버에서 전달된 Json Data 를 컴포넌트에 바로 매핑하여 사용할 수 있어 연결이 간편합니다.

  • 내부 컴포넌트간에는 강력한 연계로 웹 개발시 처리해야 할 코드를 줄였습니다.

    - SBGrid 내 Column Type 과 컴포넌트의 데이터 연계시 get/set 작업이 불필요합니다.

    - input/radio button, Master-Detail-.. 의 Combo Box 형태는 속성 추가로 데이터 연결이 됩니다.

    - Accordion Panel / Carousel 내에 Contents 를 SBUx 의 컴포넌트로 복합적으로 사용 가능합니다.

  • E-Gov 및 Eclipse IDE 개발자를 위한 Plug in 을 지원합니다

    - SBUx Tag 생성 시 코드 자동완성 기능을 제공합니다.

    - javascript 를 이용한 SBUx API 사용 시 자동완성 기능을 제공합니다.
