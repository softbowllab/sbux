SBUx v2.6

HTML 만 알고 있으면 쉽게 개발할 수 있는 UI/UX 컴포넌트

Get Documents

SBUx v2.6 Components

SBUx v2.6 는 SI 및 Application 개발자들이 퀄리티 있는 화면을 만들기 위해 필요한 다양한 기능등을 쉽게 사용할 수 있도록 지원하는 UI/UX 컴포넌트입니다.

IE, 크롬, 파이어폭스, 사파리등의 PC 브라우저와 HTML5가 지원되는 최신의 모바일 환경(스마트폰, 태블릿PC)에서 사용이 가능합니다.

Browser Support

크롬, 파이어폭스, 사파리, IE 10 이상을 권장합니다.

Getting Started

  • 설정

    SBUx 구동을 위한 초기값 설정을 합니다.

    {% highlight html %} var SBUxConfig = { License : "SUE20-2095-17644-316", Path : "/resources/SBUx/", Locale : "ko", // { ko | en | ja } Theme : "bootstrap", // { none | bootstrap } CustomCSS : "SBUxCustom_Demo.css", DefaultSetFile : "SBUxDefault.js", Components : "input,button", // 사용하는 컴포넌트만 DownloadType : "manual", // manual, pttgc, ScopeType : "manual", // angularJS 의 apply() 를 마지막에 수동으로 실행하기 위한 부분인데. 필요없다. Debug : true, DeveloperTipType : "none", // { console | alert | storage | none } SystemLogType : "console", { console | storage | none} Animation : true, Design : true, // css 를 호출, false 시 css 호출하지 않음. SupportVersion : "2.0", // "2.0" : SBUx2.0 버전 형태 , (default) "2.5" : SBUx 2.5 버전형태 LoadingType : "holding", // none | holding | progress (default) "holding" LoadIncludeClass 또는 LoadClass : 'lazyLoadClass', // 선언한 Class 가 있는 컴포넌트만 우선 로드 LoadExcludeClass : 'notLoadClass', // 선언한 Class 를 제외한 컴포넌트 로드 CustomAttrs : ['custom-attr1','custom-attr2'], // 받아들이는 custom 속성에 대한 정의 // UsingMemory : true, SBGrid : { Theme : 'default', DefaultSetFile : 'SBGridDefault.js', // { default | skyblue | oceanblue | bluishgreen | lightgreen | // graypurple | pinkgray | darkgray | blackgray | claret | charcoalgray } Version2_1 : true, Version2_5 : false }, SBChart : { Version2_0 : true } }; {% endhighlight %}

Features

  • SVG(Scalable Vector Graphics) 기반이기 때문에 해상도 제한이 없으며 리사이징되어도 깨지지 않습니다.
  • 일반적이고 간편하게 차트를 구현할 수 있습니다.
  • D3.js V4+ 기반이며 완벽한 수준의 dependency Injection을 제공합니다.
