## global.size
---

차트의 높이와 너비를 지정합니다.

**TODO** 이 옵션을 지정하지 않으면 첨부 된 상위 요소의 크기에 따라 차트 너비가 계산됩니다.

`#2.0.0`

* Type : object

* Format
```javascript
global: {
	size: { 
		width:800, height:400
	}
}
```

### global.siz.width 
차트 너비를 지정 지정합니다.

`#2.0.0`

* Type : number

* Default : 0

### global.siz.height
차트 높이를 지정 지정합니다.

`#2.0.0`

* Type : number

* Default : 0
