wrap-style
===
---

## Discription
Component의 최상단 root에 CSS를 지정합니다.

`#2.0.0`

---

## Value

### wrap-style="compWrapStyle"

* compWrapStyle : 적용할 CSS 구문에서 '{' , '}'를 제외한 값
* Value Type : string
* Default : X

---
## Example
{% highlight html %}
<sbux-input id="inputIdx" name="inputIdx" uitype="text" wrap-style="font-weight: bold;"></sbux-input>
{% endhighlight %}
