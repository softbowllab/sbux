style
===
---

## Discription
Component의 CSS를 지정합니다.

`#2.0.0`

---

## Value

### style="compStyle"

* compStyle : 적용할 CSS 구문에서 '{' , '}'를 제외한 값
* Value Type : string
* Default : X

---
## Example
{% highlight html %}
<sbux-input id="inputIdx" name="inputIdx" uitype="text" style="width:100px;"></sbux-input>
{% endhighlight %}
