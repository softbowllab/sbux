## data.legends
---

범례별 이름을 지정합니다. 

 Series의 Key별로 지정할 범례명을 지정(예:{'키':'범례'}) 합니다. 이때 지정되지 않은 Series는 Key값을 그대로 사용합니다.


`#2.0.0`

* Type : object

* Format
```javascript
data : {
	legends: {'2015': '15년', '2015': '16년', '2015': '17년'}
}
```
