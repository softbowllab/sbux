## axes.x
---

x축관련 값들을 설정합니다.

x축이 없는 원(pie), 도녓(donut) 계열 차트는 지원하지 않습니다.

`#2.0.0`

* Type : object

* Format
```javascript
axes: {
	x : {show:false}
}
```

### axes.x.show

x축의 show/hidden 을 지정합니다.

`#2.0.0`

* Type : boolean

* Default : true
