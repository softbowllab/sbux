## axes.y
---

y축 관련 값들을 설정합니다.

y축이 없는 원(pie), 도녓(donut) 계열 차트는 지원하지 않습니다.

`#2.0.0`

* Type : object

* Format
```javascript
axes: {
	y : {show:false}
}
```

### axes.y.show

y축의 show/hidden 을 지정합니다.

`#2.0.0`

* Type : boolean

* Default : true
